# Data and Code Availability

This integrated package contains the final synthesis paper, final manuscript source, final integrated figures, final integrated tables, and method summary materials. Full component-level code and data archives are not duplicated in this repository. They are linked through the component package DOI records listed in the root README.

Paper-level DOI records identify manuscripts. Package-level DOI records identify associated reproducibility archives. These DOI records are intentionally distinct so that manuscript versions and reproduction materials can be cited separately.

The component studies use the Zandbergen-Landini EVA transcription (ZL3b) of the Voynich Manuscript. Redistribution and reuse of transcription data should follow the original source terms.
